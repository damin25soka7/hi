# 처음 사용자를 위한 간단한 설명서

## 🎯 3단계로 시작하기

### 1단계: Node.js 설치 (한 번만 필요)

Node.js가 설치되어 있지 않다면:
- Windows/Mac: https://nodejs.org 에서 다운로드
- 설치 후 컴퓨터 재시작

### 2단계: 설정 마법사 실행

아래 방법 중 **하나**를 선택하세요:

**방법 A - 자동 설정 (추천):**
1. `setup.js` 파일을 더블클릭
2. 질문에 답하기 (Enter 누르면 기본값 사용)
3. 완료!

**방법 B - 명령어로 설정:**
```bash
npm run setup
```

### 3단계: 서버 시작

**가장 쉬운 방법 - 더블클릭:**
- Windows: `start.bat` 파일을 더블클릭
- Mac/Linux: `start.sh` 파일을 더블클릭

**또는 명령어로:**
```bash
npm start
```

**또는 npx로:**
```bash
npx .
```

## ✅ 서버가 실행되면

서버 주소: `http://127.0.0.1:32769` (기본값)

챗봇 설정에 다음을 입력:
```json
{
  "searxng-enhanced": {
    "url": "http://127.0.0.1:32769",
    "type": "http",
    "method": "sse"
  }
}
```

## 🔧 설정 변경하기

`setup.js`를 다시 실행하거나, `.env` 파일을 직접 수정하세요:

```env
SEARXNG_BASE_URL=http://localhost:32768  # SearXNG 주소
PORT=32769                                # 서버 포트
DESIRED_TIMEZONE=Asia/Seoul               # 시간대
```

## ❓ 문제 해결

### "Node.js를 찾을 수 없습니다"
→ Node.js를 설치하세요: https://nodejs.org

### "Python을 찾을 수 없습니다"
→ Python 3.9 이상을 설치하세요: https://python.org

### "SearXNG 연결 실패"
→ SearXNG가 실행 중인지 확인:
```bash
curl http://localhost:32768/search?q=test&format=json
```

### 포트가 이미 사용 중
→ `.env` 파일에서 PORT를 다른 번호로 변경 (예: 8080)
